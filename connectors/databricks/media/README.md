# Media Directory

This directory should contain the Databricks logo/icon file.

## Required File

- `databricks-48.png` - 48x48 pixel PNG icon for the Databricks connector

## Getting the Logo

You can obtain the Databricks logo from:
- Databricks brand assets: https://www.databricks.com/company/press
- Or create a 48x48 pixel PNG icon with the Databricks logo

The file should be named `databricks-48.png` and placed in this directory.
