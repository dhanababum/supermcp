# Media Directory

This directory should contain the Snowflake logo/icon file.

## Required File

- `snowflake-48.png` - 48x48 pixel PNG icon for the Snowflake connector

## Getting the Logo

You can obtain the Snowflake logo from:
- Snowflake brand assets: https://www.snowflake.com/brand-guidelines/
- Or create a 48x48 pixel PNG icon with the Snowflake logo

The file should be named `snowflake-48.png` and placed in this directory.
